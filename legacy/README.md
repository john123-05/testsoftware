# Legacy Reference

This folder contains selected old scripts copied from the inspected PC so future
agents and engineers can compare behavior.

Do not run these files as the main system. They are here for archaeology:

- `uploader/uploader.py`: old direct Supabase Storage uploader
- `uploader/uploader_combined.py`: old combined bridge/upload/log collector
- `uploader/speed_bridge.py`: old filename speed bridge
- `uploader/uploader_combined.env.example`: old non-secret example config

The real `.env` file was intentionally not copied.
